PDF Editor Pro — Editor de PDF online gratis

Edita archivos PDF directamente en tu navegador: modifica texto conservando la fuente original, añade imágenes, firma, rellena formularios, dibuja y resalta. 100% gratis, sin registro y sin subir tus documentos a la nube.

Características principales

  • Detección inteligente de texto e imágenes editables.
  • Edición de texto con preservación de fuente, tamaño, color, negrita y cursiva.
  • Herramientas de dibujo: rectángulos, elipses, líneas, flechas, lápiz y resaltador.
  • Navegación multi-página con miniaturas y zoom ajustable.
  • Deshacer y rehacer con atajos de teclado.
  • Exportación a PDF listo para descargar.
  • Servidor MCP (Model Context Protocol) para automatización desde Claude, Cursor o Windsurf.

¿Cómo editar un PDF online?

  1. Arrastra y suelta un archivo PDF en esta página (o pulsa Abrir).
  2. Pulsa Escanear para detectar el contenido editable.
  3. Edita texto, añade imágenes, dibuja o anota con las herramientas del lateral.
  4. Exporta el resultado como un nuevo archivo PDF.
